## ✨ Features

### 🧑 Player Mods
- **WalkSpeed Slider** — Adjust movement speed (1 - 500)
- **JumpPower Slider** — Adjust jump height (1 - 500)
- **Infinite Jump** — Jump endlessly without touching the ground
- **Fly Mode** — Free fly with WASD + Space/Shift controls
- **Fly Speed** — Customize flight velocity (1 - 300)
- **Noclip** — Walk through walls and obstacles
- **Reset Character** — Instant respawn
- **Full Heal** — Restore health to maximum

### 👁️ Visuals
- **Player ESP** — See players through walls with distance tracking and green highlights
- **Full Bright** — Remove darkness and shadows from any game
- **Custom Crosshair** — Green centered crosshair for better aim
- **Remove Effects** — Clear all post-processing effects (Bloom, Blur, Color Correction, etc.)

### 🗺️ Teleport
- **Click Teleport** — Hold `Ctrl` + Click to teleport to mouse position
- **Teleport to Random Player** — Instantly warp to a random player in the server
- **Teleport to Specific Player** — Enter exact username to teleport to them
- **Save Position** — Bookmark your current location
- **Load Position** — Return to your saved location anytime

### ⚙️ Utilities
- **Anti-AFK** — Prevent getting kicked for being idle
- **Rejoin Server** — Rejoin the exact same server instance
- **Server Hop** — Join a different server with available slots
- **Low Graphics Mode** — Reduce lag by forcing Plastic materials and removing textures
- **Developer Console** — Open/Close Roblox Console directly from the UI (`F9` shortcut supported)
- **Auto-Open Console** — Optionally open console automatically on script load

### 🎨 UI
- **Modern WindUI Design** — Clean, animated, and responsive interface
- **Avatar Welcome** — Personalized greeting with your Roblox avatar
- **Custom Themes** — Dark, Light, Rose, Plant, Indigo, Sky, Violet, Amber
- **Keybind Toggle** — Press `RightShift` to show/hide the UI instantly
- **Notifications** — Beautiful toast notifications for all actions

## 📥 Installation

### Method 1: Direct Load (Recommended)
```lua
loadstring(game:HttpGet("YOUR_RAW_GITHUB_LINK_HERE"))()
```

### Method 2: Manual Copy
1. Copy the script from the repository
2. Paste it into your Roblox executor
3. Press **Execute**

## 🚀 Usage

```
1. Inject your executor into Roblox
2. Paste the GreenZx Hub script
3. Press Execute
4. Press RightShift to toggle the UI
5. Navigate through tabs using the sidebar
6. Enjoy!
```

### Console Tab
- Click **"Open Console"** to launch the Roblox Developer Console
- Uses `StarterGui:SetCore("DevConsoleVisible", true)` internally
- Click **"Close Console"** to hide it
- Enable **"Auto-Open Console"** to open it automatically on load

### Fly Controls
| Key | Action |
|-----|--------|
| `W` | Fly Forward |
| `S` | Fly Backward |
| `A` | Fly Left |
| `D` | Fly Right |
| `Space` | Fly Up |
| `LeftShift` | Fly Down |

### Teleport Controls
| Input | Action |
|-------|--------|
| `Ctrl + Left Click` | Click Teleport to cursor position |
